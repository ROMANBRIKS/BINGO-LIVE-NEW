# VIP and Noble System Mapping

This file serves as the "memory" for the finalized VIP and Noble system structure, mapping the 22 Noble titles to the 6 VIP tiers and their corresponding level ranges.

## VIP Tier Structure & Noble Titles

### VIP 1 (Levels 1–9)
*   **Knight / Dame** (LV.3)
*   **Viscount / Viscountess** (LV.6)
*   **Earl / Countess** (LV.9)

### VIP 2 (Levels 10–29)
*   **Marquis / Marchioness** (LV.10)
*   **Baron / Baroness** (LV.15)
*   **Viscount Elite / Viscountess Elite** (LV.20)
*   **Earl Elite / Countess Elite** (LV.25)

### VIP 3 (Levels 30–49)
*   **Duke / Duchess** (LV.30)
*   **Grand Duke / Grand Duchess** (LV.40)
*   **Archduke / Archduchess** (LV.49)

### VIP 4 (Levels 50–59)
*   **Prince / Princess** (LV.50)
*   **Crown Prince / Crown Princess** (LV.55)
*   **King / Queen** (LV.59)

### VIP 5 (Levels 60–89)
*   **Emperor / Empress** (LV.60)
*   **Great Emperor / Great Empress** (LV.67)
*   **Legendary Emperor / Legendary Empress** (LV.74)
*   **Supreme Emperor / Supreme Empress** (LV.81)
*   **Overlord / Overlady** (LV.89)

### VIP 6 (Levels 90–119)
*   **Demi-God / Demi-Goddess** (LV.90)
*   **God of War / Goddess of War** (LV.100)
*   **Celestial God / Celestial Goddess** (LV.110)
*   **Global God / Global Goddess** (LV.119)
